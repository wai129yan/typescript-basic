📌 1. String များအတွက် Function များ
Function	ရည်ရွယ်ချက် (မြန်မာ)	ဥပမာ
toUpperCase()	စာလုံးအားလုံးကို ဗျည်းကြီးအဖြစ်ပြောင်း	'aung'.toUpperCase() → 'AUNG'
toLowerCase()	စာလုံးအားလုံးကို ဗျည်းသေးအဖြစ်ပြောင်း	'AUNG'.toLowerCase() → 'aung'
includes()	စာကြောင်းထဲမှာ string တစ်ခုပါ/မပါ စစ်	'hello'.includes('e') → true
replace()	စာသားတစ်ခုကို အခြားတစ်ခုနဲ့အစားထိုး	'hello'.replace('h', 'y') → 'yello'
slice()	စာလုံးတစ်စိတ်တစ်ပိုင်း ခြားထုတ်	'hello'.slice(1, 4) → 'ell'
📌 2. Array များအတွက် Function များ
Function	ရည်ရွယ်ချက်	ဥပမာ
push()	Array ထဲအဆုံးမှာ item ထည့်	arr.push(5)
pop()	Array အဆုံးက item ဖျက်	arr.pop()
shift()	Array အအစက item ဖျက်	arr.shift()
unshift()	Array အစမှာ item ထည့်	arr.unshift(1)
map()	Array ထဲ item တစ်ခုချင်းကို ပြောင်း	arr.map(x => x * 2)
filter()	condition တစ်ခုအပေါ်မူတည်ပြီး item ရွေး	arr.filter(x => x > 5)
reduce()	Array ကို တစ်ခုတည်းသောတန်ဖိုးအဖြစ်ချုပ်	arr.reduce((a, b) => a + b, 0)
find()	condition ကိုလက်ခံတဲ့ ပထမ item ကို return	arr.find(x => x > 3)
includes()	Array ထဲမှာ တန်ဖိုးတစ်ခုပါ/မပါ စစ်	[1,2,3].includes(2) → true
📌 3. Number များအတွက် Function များ
Function	ရည်ရွယ်ချက်	ဥပမာ
toFixed()	သတ်မှတ်ထားတဲ့ ဒဿမနဲ့ပြ	(12.345).toFixed(2) → '12.35'
parseInt()	string ကို integer ပြောင်း	parseInt('123') → 123
parseFloat()	string ကို float ပြောင်း	parseFloat('12.34') → 12.34
isNaN()	NaN ဖြစ်/မဖြစ် စစ်	isNaN('abc') → true
📌 4. Math Function များ (Global Math Object)
Function	ရည်ရွယ်ချက်	ဥပမာ
Math.round()	အနီးဆုံးလုံးပြတ်	Math.round(4.6) → 5
Math.floor()	အောက်ဆုံးလုံးပြတ်	Math.floor(4.9) → 4
Math.ceil()	အပေါ်ဆုံးလုံးပြတ်	Math.ceil(4.1) → 5
Math.random()	0 နဲ့ 1 အကြားက random number	Math.random()
Math.max()	အကြီးဆုံးကို ရွေး	Math.max(1, 5, 3) → 5
Math.min()	အငယ်ဆုံးကို ရွေး	Math.min(1, 5, 3) → 1
📌 5. Date Function များ (Date Class)
Function	ရည်ရွယ်ချက်	ဥပမာ
new Date()	ယနေ့ရက်စွဲ object ဖန်တီး	new Date()
getFullYear()	နှစ်ကို ယူ	date.getFullYear()
getMonth()	လ (0-based) ယူ	date.getMonth()
getDate()	ရက် (day of month) ယူ	date.getDate()
toISOString()	ISO format string	date.toISOString()
📌 6. JSON Function များ
Function	ရည်ရွယ်ချက်	ဥပမာ
JSON.stringify()	object → JSON string	JSON.stringify({a:1}) → '{"a":1}'
JSON.parse()	JSON string → object	JSON.parse('{"a":1}') → {a:1}
📌 7. Other Useful Global Functions
Function	ရည်ရွယ်ချက်	ဥပမာ
setTimeout()	အချိန်ကြာပြီး callback run	setTimeout(() => {}, 1000)
setInterval()	ကြာချိန်တိုင်း callback run	setInterval(() => {}, 1000)
alert()	message တစ်ခုပြ	alert('Hello')
console.log()	Console ထဲပြ	console.log("Hello")
